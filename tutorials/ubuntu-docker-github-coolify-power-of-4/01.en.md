### Q: Can I deploy multiple apps on the same server?

**A:** Absolutely! Coolify excels at running multiple applications. Each app gets:
- Its own container
- Unique domain/subdomain
- Isolated environment
- Independent deployment pipeline

### Q: How do I rollback to a previous version?

**A:** In Coolify:
1. Go to **Deployments** tab
2. Find the previous successful deployment
3. Click **Rollback**

### Q: What are the server requirements?

**A:** Minimum requirements:
| Resource | Minimum | Recommended |
|----------|---------|-------------|
| CPU | 1 vCPU | 2+ vCPU |
| RAM | 2 GB | 4+ GB |
| Storage | 20 GB | 40+ GB SSD |

### Q: Can I use GitLab instead of GitHub?

**A:** Yes! Coolify supports:
- GitHub
- GitLab  
- Bitbucket
- Gitea
- Any Git repository via URL

### Q: How do I update Coolify?

**A:** 
```bash
curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash
```
